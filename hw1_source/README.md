### How to use

1. Use the toolbar to choose tools: brush, eraser, text, image.
2. Adjust color and brush size before drawing.
3. Use layer panel to create/switch/reorder layers.
4. Click refresh/clear button to reset canvas when needed.
5. Click download button to export your artwork.
